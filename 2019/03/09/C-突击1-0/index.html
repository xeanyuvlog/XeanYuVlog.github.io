C++是什么语言?

面向对象编程语言，继承了C的特点，由Bjarne Stroustrup开发，据说是为了让编程变得更快乐，虽然我也没感受的到快乐，可能这就是大牛和萌新的区别吧，但是相比C来说，的确快乐一些。
C++ 在C的基础上进行了扩充和完善
C++ 的应用范围偏底层偏后台，比如游戏后台，游戏开发，移动端APP都可以用C++ 进行开发。

我学C++ 主要还是OI现在基本主角是C++。

include的注意点

还要注意C的库和C++的库

其中的区别在于后缀名.h
第一个引入是C++的库，第二个是C语言的库，这里要注意的是，C++的String和C语言的String.h的库没有半毛钱关系。在C++中，C的库都会去掉.h，在文件头加上字符c
就像这样#include <cstring> 其中的cstring和string.h才是相对应的。
